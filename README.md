# Hungry Cat

A sushi-munching, rat-dodging Pac-Man clone starring a hungry cat. Built with vanilla HTML/CSS/JS and the Canvas API.

## How to Play

Guide your tabby cat through the maze, eating every piece of sushi to clear the level. Avoid the rats — or grab a wasabi power-up to turn the tables and chase them down for bonus points.

### Controls

| Key | Action |
|---|---|
| Arrow keys / WASD | Move |
| Enter / Space | Start / Restart |

### Scoring

| Event | Points |
|---|---|
| Sushi eaten | 10 |
| Rat eaten (wasabi mode) | 200 |
| Level clear bonus | 500 |

## Running Locally

Open `index.html` in a browser. No build step required.

## Deployment

Static files only — deploy directly to GitHub Pages.
